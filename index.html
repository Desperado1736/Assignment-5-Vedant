How to Apply CSS to HTML?

One way to include an external CSS file in an HTML file is by using the <link> element within the <head> section of your HTML document. Here's an example:

    
      <!DOCTYPE html>
      <html>
      <head>
        <title>My Website</title>
        <!-- Include external CSS file -->
        <link rel="stylesheet" type="text/css" href="styles.css">
      </head>
      <body>
        <p>This is a paragraph with custom CSS styling.</p>
      </body>
      </html>

How to Use JavaScript in a Webpage?

To execute a JavaScript script on your webpage, simply employ the <script> element with the src attribute indicating the script file's location.. Here's an example:

    
      <script src="script.js"></script>

To write JavaScript code within an HTML file, you can use the <script> element directly within the <body> or <head> section. Here's an example:

    
      <script>
        // JavaScript code here
        document.write("This is JavaScript code within an HTML file.");
      </script>
